========================================
🔨 如何构建
========================================

前提条件：
  - 已安装 Java 21 或更高版本
  - 已安装 Maven 3.6 或更高版本

构建命令：
  mvn clean package

或者使用你已安装的Maven：
  E:\MCPLUGINS\apache-maven-3.9.11\bin\mvn clean package

构建完成后：
  插件文件: target\MumuCustomization-1.0.0.jar

========================================
✨ 安装使用
========================================

1. 构建插件
   mvn clean package

2. 复制插件
   从: target\MumuCustomization-1.0.0.jar
   到: <服务器>\plugins\

3. 启动服务器
   配置文件自动生成在:
   plugins/MumuCustomization/config.yml

4. 自定义配置
   编辑配置文件，使用 /mumu reload 重载
